# 番茄短篇大女主大纲

## 工作流程

1. 先确认目标产物：`选题方向`、`开篇节奏分析`、`全文大纲`、`开篇钩子`、`分段细纲` 中选择一种；如果用户已经说清楚，就直接执行。
2. 生成选题、节奏分析或大纲前，先阅读 `references/fanqie-short-danvzhu-patterns.md`，把输出锚定在本地短篇大女主样本的结构，而不是泛泛写女强爽文。
3. 明确样本边界：本地样本只包含开头到中前段，不能把样本当完整全文结局；制作全文大纲时要在样本开篇机制之外，自行补足中后段升级、审判和结局闭环。
4. 先定一句话核心不公：谁用亲情、婚姻、身份、制度、性别规训或信息差压迫女主/主角，制造了什么可被公开清算的损失。
5. 再定主角优势：重生记忆、真实身份、股份/法律文件、专业能力、时代信息、技术能力、财富、舆论、规则理解、强行动力，只选最能支撑反击的一条主轴。
6. 制作大纲时按短篇分段推进，不要写成长篇卷纲；每一段都要有明确爽点功能：受辱、识破、掀桌、取证、反击、升级、公开、清算、换生活。

## 输出规则

- 除非用户另有要求，否则全部使用中文。
- 优先输出可直接写正文的内容，不写空泛理论。
- 开篇必须在 300 字内抛出强钩子：假离婚秒速签字、被克扣生活费、被偷录取通知书、金主出事后金丝雀上桌、办公室被绿茶男占领、女性想找“男老婆”等。
- 大女主爽点必须具体到动作和证据，不只写“女主变强”：要写离婚证、银行流水、录取信息、持股证明、遗嘱、审计报告、营业执照、合同、监控、热搜、法律文件。
- 主角不要只靠别人救场；可以借势，但关键反击必须由主角主动完成：签字、查账、进股东会、公开证据、创业、换城市、报警、起诉、辞退、收权。
- 反派不能只嘴坏；要让其错误行为有可清算后果：名誉崩塌、权力失效、亲友倒戈、财产损失、职业处分、法律责任、失去主角供养或情感位置。
- 不要照搬样本人名、设定、标题、标志性桥段或具体句子，只提取节奏机制。
