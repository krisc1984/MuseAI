# 番茄短篇真假千金大纲

## 工作流程

1. 先确认目标产物：`选题方向`、`剧情节奏分析`、`全文大纲`、`开篇钩子`、`分段细纲` 中选择一种；如果用户已经说清楚，就直接执行。
2. 生成选题、节奏分析或大纲前，先阅读 `references/fanqie-short-qianjin-patterns.md`，把输出锚定在本地短篇真假千金样本的结构，而不是泛泛写豪门真假千金。
3. 先定一句话身份错位：谁被误认为占了谁的位置，谁真正掌握家庭、学校、资源或舆论规则。
4. 再定反套路爽点：主动退位、咸鱼不争、公开系统、傻白甜被反派家人保护、管家女儿用事实打脸、真千金误判豪门规则，只选最能制造开篇追读的一条主线。
5. 制作大纲时按短篇分段推进，不要写成长篇卷纲；每一段都要有明确功能：身份揭露、站队试探、茶言茶语、误判加深、证据反打、家人醒悟、公开审判、余韵收束。

## 输出规则

- 除非用户另有要求，否则全部使用中文。
- 优先输出可直接写正文的内容，不写空泛理论。
- 必须提醒自己：本地样本只是不完整开头，不是完整全文；总结样本时写“开头样本大纲”，制作新故事时可以基于开头机制合理推演完整短篇闭环。
- 开篇必须在 300 字内抛出身份冲突：真千金回家、假千金被赶、系统预告、管家女儿被扣帽子、全家反派审茶、疯院归来、学校公开羞辱等。
- 身份冲突必须具体到场景和证据，不只写“真千金回来后被偏爱”：要写房间归属、餐桌座位、亲子鉴定、学籍转入、家族会议、佣人房、新闻热搜、同学证词、监控或公开发言。
- 女主不要只靠受委屈获得爽感；要让她至少拥有一种能完成反击的力量：咸鱼心态、事实证据、父母/兄妹反向站队、职业/学习能力、系统信息差、朋友同学、家族规则、精神状态反差。
- 茶味角色不能只会哭；要让她每次挑衅都推动一个新误判，并被更硬的事实、更强的规则或更会演的人反噬。
- 不要照搬样本人名、设定、标题、标志性桥段或具体句子，只提取节奏机制。
